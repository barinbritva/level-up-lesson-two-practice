# Запуск проекта

## Основная сборка
```bash
# установить зависимости
npm install
# собрать скрипты, стили, картинки
npm run build

# запустить сборку в режиме watch
npm run build:dev
```

## Для ознакомления

```
# Cборка через webpack (только скрипты)
npm run wp-build
npm run wp-build:dev
```

```
# Cборка через tsc (только скрипты)
npm run tsc-build
npm run tsc-build:dev
```
